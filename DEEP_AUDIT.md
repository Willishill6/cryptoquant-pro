# 深度审查 — 缺失功能清单

## 核心缺失功能（按优先级排序）

### 1. AI自动调度引擎 ⭐⭐⭐
- 系统缺少根据胜率/准确率自动切换主力AI模型的机制
- 需要：调度引擎Hook + 调度面板 + 切换日志 + 权重自动调整
- 位置：AI系统页面新增"智能调度"Tab

### 2. 模型融合集成信号 ⭐⭐⭐
- A/B测试面板只展示单模型对比，缺少融合信号
- 需要：加权投票融合 + 集成准确率对比 + 融合信号展示
- 位置：A/B测试面板底部新增融合信号区域

### 3. 综合资金曲线/权益曲线 ⭐⭐⭐
- Dashboard的RealtimePnL只有简单盈亏，缺少完整的权益曲线
- 需要：30天权益曲线 + 最大回撤标注 + 基准对比(BTC) + 夏普比率走势
- 位置：Dashboard新增或替换现有盈亏区域

### 4. 风控联动告警横幅 ⭐⭐
- 状态栏有告警数，但缺少全局风控联动
- 需要：当API延迟>50ms/WS断连/回撤超限时，全局红色警告横幅
- 位置：DashboardLayout顶部

### 5. 交易信号实时推送面板 ⭐⭐
- AI系统产生信号但缺少统一的信号推送/通知面板
- 需要：实时信号流 + 信号强度 + 一键跟单
- 位置：Dashboard或交易引擎页面

### 6. 模型性能排行榜（实时） ⭐⭐
- 各页面分散展示模型数据，缺少统一实时排行
- 需要：综合评分排行 + 趋势箭头 + 历史排名变化
- 位置：AI系统"智能调度"Tab内

## 开发计划
1. useAIScheduler Hook — 调度引擎核心逻辑
2. AISchedulerPanel 组件 — 调度面板UI
3. 融合信号系统 — 集成到A/B测试面板
4. 全局风控横幅 — 集成到DashboardLayout
5. 权益曲线增强 — 集成到Dashboard
6. 信号推送面板 — 集成到Dashboard
