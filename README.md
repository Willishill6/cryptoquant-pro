# CryptoQuant Pro

AI驱动的全币种量化交易系统 · 自我学习 · 持续进化 · 覆盖441+币种 · 5年+历史数据

## 技术栈

- **前端**: Vite + React + TypeScript + TailwindCSS
- **后端**: Node.js + Express + WebSocket
- **数据**: 币安实时行情 WebSocket 代理

## 本地运行

```bash
# 安装依赖
npm install

# 开发模式
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
NODE_ENV=production node dist/index.js
```

## 功能特性

- 实时行情推送（441+币种）
- AI 量化策略系统
- 多因子 Alpha 系统
- AI 进化实验室
- 风险管理系统
- 数据分析与监控
